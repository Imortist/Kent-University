Write a VM to Hack  assembler translator, as described in Chapter 7 of nand2tetris book.
Base java program is provided.

Tasks:
> push constant
> pop to register D
> The add and sub instructions
> The and and or instructions
> The neg and not instructions
> the eq,lt, and gt instructions
> push local/argument/this/that
> pop  local/argument/this/that
> push and pop of temp/pointer
> push/pop static
